Atajos Generales

  • Paleta de comandos: CTRL + SHIFT + P / F1
  • Explorador de archivos: CTRL + P
  • Nueva Instancia(Ventana): CTRL + SHIFT + N
  • Cerrar Instancia(Ventana): CTRL + W
  • Nueva Pestaña: CTRL + N
  • Mostrar/Ocultar Sidebar(Explorador): CTRL + B (CTRL + SHIFT + E)

Multicursor

  • Insertar multicursor con mouse: ALT + CLICK
  • Insertar Cursor con teclado:
    • Arriba: CTRL + ALT + ↑
    • Abajo: CTRL + ALT + ↓
  • Deshacer Cursor: CTRL + U (Para borrar el último cursor que se haya agregado, se puede usar múltiples veces, hasta solo tener uno.)

Selección

  • Seleccionar todo el texto CTRL + A
  • Seleccionar Línea Actual: CTRL + L
  • Seleccionar todas las ocurrencias de la selección actual: CTRL + SHIFT + L
  • Buscar y seleccionar las ocurrencias de la selección actual una por una: CTRL + D
  • Expandir/reducir selección: SHIFT + ALT + ←/→
  • Seleccion de columnas con mouse SHIFT + ALT + arrastrar
  • Seleccion de columnas con teclado SHIFT + ALT + ↑/↓/←/→

Manejo del Código

  • Mover línea: ALT + ↑/↓
  • Copiar línea arriba o abajo: SHIFT + ALT + ↑/↓
  • Borrar línea: CTRL + Shift + K
  • Insertar línea:
    • Arriba: CTRL + SHIFT + ENTER
    • Abajo: CTRL + ENTER
  • Navegar al final del fragmento: CTRL + SHIFT + \
  • Mover indexación:
    • Indexar derecha: CTRL + ]
    • Indexar izquierda: CTRL + [
  • Ir al inicio/final de un archivo: CTRL + Home/End
  • Comentar línea: CTRL + /
  • Comentar selección: CTRL + K CTRL + C
  • Remover comentario de selección: CTRL + K CTRL + U
  • Comentar/descomentar bloque: SHIFT + ALT + A
  • Formatear documento: SHIFT + ALT + F
  • Formatear selección CTRL + K CTRL + F
  • Arreglo rápido: CTRL + . (Permite ver los diferentes tips que te da VSCode respecto al código actual.)

Manejo del Editor

  • Dividir editor: CTRL + \
  • Moverse entre los grupos de pantallas: CTRL + 1 / 2 / 3(También se puede usar CTRL + K + CTRL + ←/→)
  • Cambiar ubicación de las pantallas: SHIFT + ALT + 0
  • Pantalla completa: F11
  • Mostrar explorador de archivos: CTRL + SHIFT + E
  • Mostrar búsqueda: CTRL + SHIFT + F
  • Mostrar debug: CTRL + SHIFT + D
  • Mostrar extensiones: CTRL + SHIFT + X
  • Modo Zen (Concentracion): CTRL + K Z

Búsqueda y Remplazo

  • Buscar: CTRL + F
  • Reemplazar: CTRL + H
  • Siguiente/anterior búsqueda: F3 / SHIFT + F3
  • Seleccionar todas las ocurrencias de una búsqueda: ALT + ENTER
  • Busqueda en todos los archivos: CTRL + SHIFT + F
  • Reemplazar en todos los archivos: CTRL + SHIFT + H

Navegación

  • Ir a una línea específica: CTRL + G
  • Ir a un bloque: CTRL + SHIFT + O
  • Ver el panel de problemas: CTRL + SHIFT + M

Manejo de Archivos

  • Abrir archivo o proyecto: CTRL + O
  • Guardar: CTRL + S
  • Guardar todo: CTRL + K S
  • Cerrar todo: CTRL + K CTRL + W
  • Abrir la última pestaña cerrada: CTRL + SHIFT + T
  • Moverse entre archivos: CTRL + TAB / CTRL + SHIFT + TAB
  • Copiar ruta actual: CTRL + K P
  • abrir archivo actual en el finder/explorador: CTRL + K R
  • Abrir el archivo actual en una ventana/instancia: CTRL + K O

Extra

  • Mostrar/ocultar terminal: CTRL + `
  • Crear nueva terminal: CTRL + SHIFT + `
  • Configuracón del editor: CTRL + ,
  • Ver atajos de teclado: CTRL + K CTRL + S
  • Expandir/Contraer Bloques: Ctr + K + L
  • Contraer todos los bloques: CTRL + K + 0
  • Expandir todo los bloques: CTRL + K + J
  • Deshacer Acción: CTRL + Z
  • Rehacer Acción: CTRL + Y
